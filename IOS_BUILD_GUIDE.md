# 🍎 دليل بناء iOS

## 🚀 كيفية بناء IPA:

### 1️⃣ عبر GitHub Actions:
1. اذهب إلى: Repository → Actions
2. اختر: "🍎 Build iOS IPA"
3. اضغط: "Run workflow"
4. انتظر: 15-20 دقيقة
5. حمّل: sr3h-ios-app من Artifacts

### 2️⃣ التثبيت:
- استخدم AltStore أو Sideloadly
- أو Xcode للأجهزة المطورة

### 3️⃣ المتطلبات:
- iOS 12.0+
- iPhone/iPad
- مساحة 100MB

## 🎯 النتيجة:
ستحصل على تطبيق iOS كامل جاهز للاستخدام!
